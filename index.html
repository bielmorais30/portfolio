Sobre mim

Sou Gabriel Morais de Oliveira, desenvolvedor apaixonado por tecnologia e construção de soluções que resolvem problemas reais. Atualmente no 7º semestre de Ciência da Computação, já acumulei experiência prática com desenvolvimento web full stack, back-end robusto e integração com IA.

Tenho experiência com o ecossistema PHP/Laravel, desenvolvimento Java com Spring Boot (incluindo JWT, OAuth2, Docker e AWS), além de criação de bots inteligentes com JavaScript e IA generativa.

Gosto de criar aplicações que combinam boa arquitetura com funcionalidades inovadoras. Meu projeto mais recente integra o WhatsApp com LLMs para processar e interpretar mensagens de áudio e texto automaticamente.
